# LazyTFFXIV

Windows 自動登入輔助工具，一鍵完成帳號密碼與 TOTP 驗證碼輸入，省去每次手動操作的麻煩。

## 這個工具能做什麼？

啟動後，自動執行以下登入流程：

1. 開啟指定的應用程式
2. 自動輸入您的密碼
3. 自動輸入 TOTP 驗證碼（動態密碼）
4. 自動送出登入

**您只需點擊一個按鈕，即可完成整個登入過程。**

## 系統需求

- **作業系統**：Windows 7 / 8 / 10 / 11
- **執行環境**：.NET Framework 4.8（Windows 10 以上通常已內建）

## 下載與執行

1. 從 Release 頁面下載 `LazyTFFXIV.exe`
2. 直接雙擊執行，無需安裝

## 使用前準備

請先準備以下資訊：

| 項目 | 說明 |
|------|------|
| **登入密碼** | 您的帳號密碼 |
| **TOTP 密鑰** | 設定兩步驟驗證時取得的 Secret Key（Base32 格式，例如：`JBSWY3DPEHPK3PXP`） |
| **應用程式路徑** | 要自動登入的 .exe 檔案位置 |

> **如何取得 TOTP 密鑰？**
> 設定兩步驟驗證時，除了 QR Code 外，通常會顯示一組文字密鑰，這就是 Secret Key。

## 操作說明

### 首次設定

1. **設定密鑰** - 輸入您的 TOTP Secret Key
2. **設定密碼** - 輸入您的登入密碼
3. **選擇應用程式** - 選擇要自動登入的程式

當三個項目都設定完成（狀態燈皆為綠色），「執行」按鈕會亮起。

### 執行登入

點擊「執行」按鈕，程式會：
- 自動啟動應用程式
- 等待程式就緒
- 依序輸入密碼、驗證碼並送出

### 調整延遲時間

如果登入速度過快導致失敗，可調高延遲時間（預設 300 毫秒，可調整 50-2000 毫秒）。

## 資訊安全

### 加密機制說明

本工具使用 **Windows DPAPI（Data Protection API）** 加密您的敏感資料：

- 加密金鑰由 Windows 自動管理，綁定您的 **Windows 使用者帳戶**
- 只有在 **同一台電腦** 且 **同一個 Windows 帳戶** 登入狀態下才能解密
- 加密檔案即使被複製，在其他電腦或帳戶上也無法解密

### 您的資料如何被保護？

| 安全機制 | 說明 |
|----------|------|
| **加密儲存** | 密碼與密鑰使用 DPAPI 加密，絕不以明碼儲存 |
| **帳戶綁定** | 加密資料綁定您的 Windows 帳戶，其他帳戶無法解密 |
| **本機儲存** | 所有資料僅儲存於您的電腦，不會上傳至任何伺服器 |
| **隱碼輸入** | 輸入密碼時顯示 ● 符號，防止旁人窺視 |

### 資料儲存位置

所有設定儲存於：`%AppData%\LazyTFFXIV\`

| 檔案 | 內容 | 加密狀態 |
|------|------|----------|
| `secretkey.dat` | TOTP 密鑰 | 已加密 |
| `password.dat` | 登入密碼 | 已加密 |
| `config.json` | 程式路徑、延遲設定 | 未加密（非敏感資料） |

### 安全情境分析

#### ✅ 安全的情境（資料受保護）

| 情境 | 說明 |
|------|------|
| 檔案被複製到其他電腦 | 無法解密，因為缺少原始帳戶的加密金鑰 |
| 硬碟被取出接到其他電腦 | 無法解密，即使能讀取檔案內容也只是亂碼 |
| 同電腦的其他使用者帳戶 | 無法解密，DPAPI 綁定特定帳戶 |
| 電腦送修但有設定 Windows 帳戶密碼 | 維修人員無法登入您的帳戶，無法解密 |

#### ⚠️ 危險的情境（資料可能外洩）

| 情境 | 風險等級 | 說明 |
|------|----------|------|
| **他人使用您的 Windows 帳戶** | 🔴 高 | 已登入狀態下，任何程式都能解密 DPAPI 資料 |
| **電腦感染惡意軟體** | 🔴 高 | 惡意程式以您的身份執行，可直接解密資料 |
| **使用共用帳戶** | 🔴 高 | 所有共用該帳戶的人都能解密 |
| **電腦未設密碼被他人開機** | 🔴 高 | 對方可直接登入您的帳戶 |
| **遠端桌面被入侵** | 🔴 高 | 攻擊者取得您的帳戶控制權 |
| **電腦遺失或被盜** | 🟡 中 | 若 Windows 帳戶有強密碼則相對安全 |
| **Windows 帳戶密碼外洩** | 🟡 中 | 攻擊者需同時取得實體電腦存取權 |

#### ❌ 本工具無法防護的風險

- **鍵盤側錄程式**：若電腦已被安裝鍵盤側錄軟體，輸入的密碼會被記錄
- **螢幕錄影**：惡意程式可能錄製螢幕，擷取自動輸入的過程
- **記憶體讀取**：進階攻擊可從程式記憶體中擷取已解密的資料

### 安全建議

#### 基本防護（強烈建議）

- ✅ 為 Windows 帳戶設定強密碼
- ✅ 離開電腦時鎖定螢幕（Win + L）
- ✅ 不要在公用電腦或網咖使用
- ✅ 不要與他人共用 Windows 帳戶
- ✅ 定期更新 Windows 和防毒軟體

#### 進階防護（選用）

- ✅ 啟用 Windows BitLocker 全磁碟加密
- ✅ 啟用 Windows Hello 生物辨識登入
- ✅ 定期檢查已安裝的程式，移除可疑軟體
- ✅ 使用獨立的 Windows 帳戶（非管理員）執行日常工作

#### 緊急處理

如果懷疑資料已外洩：

1. 立即更改遊戲帳號密碼
2. 重新產生新的 TOTP 密鑰
3. 刪除 `%AppData%\LazyTFFXIV\` 資料夾
4. 檢查電腦是否有惡意軟體

## 常見問題

### Q：為什麼執行按鈕是灰色的？
A：請確認三個設定項目（密鑰、密碼、應用程式）都已完成，狀態燈皆顯示綠色。

### Q：登入失敗，驗證碼好像沒輸入完整？
A：請嘗試增加延遲時間，讓程式有足夠時間等待應用程式回應。

### Q：換電腦後資料還在嗎？
A：不在。加密資料綁定於 Windows 帳戶，需在新電腦重新設定。

### Q：如何清除所有資料？
A：刪除 `%AppData%\LazyTFFXIV\` 資料夾即可完全移除所有設定。
